# Felis, by Alice & Bob


## Documentation

Located in the docs folder

Install mkdocs:
```
pip install mkdocs-material
```

View the documentation locally:
```
mkdocs serve
```

To publish the documentation, create a pull request.
Once the PR is rebased, it will trigger the CI and publish
the new version.
